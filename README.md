# OPS-Systems Framework

This repository serves as a centralized management hub for various automation modules.

## Projekt-Struktur / Module
| Modul | Kurzbeschreibung |
| :--- | :--- |
| **Cross-Platform** | Automated via Jarvis System (ISO-IEC 26514 Compliant) |
| **Linux** | Automated via Jarvis System (ISO-IEC 26514 Compliant) |
| **Setup** | This module provides an automated setup script for Zsh, including the installation of essential plugins and configuration file symlinking, ensuring a consistent shell environment based on the repository's configurations. |
| **Windows-Core** | This module provides essential PowerShell functions for Windows environments, integrating a Python-based network scanner and offering a daily dashboard to monitor Git repository statuses. |

---
> All systems are managed under ISO/IEC 26514 compliant documentation standards.
