# VS-Code Module

## Overview

This README provides an overview of the settings configuration for a senior sysadmin.

### Settings File: `settings.json`

The `settings.json` file contains the default settings for various applications and tools used by this senior sysadmin. The file is structured as a JSON object, with each key-value pair representing a specific setting.

### Configuration Details

Here are some notable settings and their values:

* **Debugging**: Debug mode is set to open on debug break.
* **Editor Settings**:
	+ Tab completion is turned off.
	+ Word separators include various special characters.
	+ Font size is set to 14 points.
* **Powershell Settings**:
	+ Show panel movement buttons is disabled.
	+ Code folding shows the last line of code.
	+ Integrated console does not show a start banner and will not focus on the console after execution.
* **Security Settings**:
	+ Snyk security at inception is enabled, with auto-configure and execution frequency set to "On Code Generation".
* **Telemetry Settings**: Telemetry is disabled, except for Snyk which has its own settings.
* **Window Settings**: The system color theme is set to dark.

### Additional Notes

These settings are intended to provide a consistent experience across various applications and tools used by this senior sysadmin.
